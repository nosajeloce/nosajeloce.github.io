Projet 1: Système de réservation de billets de cinéma en ligne

Description: Mon premier projet informatique porte sur un système de réservation de billets de cinéma en ligne. Dans le cadre du cours Applications Web 1 (420-G16-RO), une compagnie fictive (CinePro) nous demande de répondre aux besoins des utilisateurs dont l'un est d'effecuter des réservations de visionnement de films à partir de leurs ordinateurs.

Ce projet a été réalisé avec les langages et les framework appris en classe. Les langages et les framework appris:

  • HTML5
  • CSS3
  • Framework BootStrap 5
  • PHP
  • Javascript
  • JQuery

Voici le lien vers ce projet: https://github.com/nosajeloce/Projet-CinePro
